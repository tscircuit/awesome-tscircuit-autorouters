# awesome-tscircuit-autorouters

Custom autorouters that can be dropped into tscircuit

- [tscircuit-autorouter](https://github.com/tscircuit/tscircuit-autorouter) - The default tscircuit autorouter, a [hypergraph-based](https://blog.autorouting.com/p/hypergraph-autorouting) autorouter
- [@tscircuit/krt-wasm](https://github.com/tscircuit/krt-wasm) - [Kicad Routing Tools](https://github.com/drandyhaas/KiCadRoutingTools) ported to be compatible with tscircuit, uses a fast Rust/WASM grid solver
